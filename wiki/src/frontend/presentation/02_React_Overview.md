# 02. React Overview

### 1. React 개념
React는 사용자 인터페이스(UI)를 구축하기 위한 JavaScript 라이브러리입니다. 전체 화면을 '컴포넌트(Component)'라는 독립적인 단위로 분할하여 개발하고, 이를 레고 블록처럼 조립하여 복잡한 UI를 구성하는 방식을 취합니다.

### 2. Virtual DOM (가상 DOM)의 원리
*   **직접 DOM 조작(Vanilla JS)**: 데이터 변경 시 브라우저의 화면 전체를 다시 그려 성능 저하와 화면 깜빡임이 발생할 수 있습니다.
*   **React의 방식**: 데이터 상태(State)가 변하면 메모리상의 '가상 DOM'에서 변경된 부분만 먼저 계산합니다. 이후 실제 브라우저 화면(DOM)에는 차이가 발생한 부분만 효율적으로 업데이트하여 앱 수준의 부드러운 사용자 경험을 제공합니다.

### 3. Vite + React 선택 이유
*   **경량화 및 속도**: 서버 사이드 렌더링(SSR) 기능이 포함된 무거운 프레임워크(Next.js) 대신, 클라이언트 사이드 렌더링(CSR)에 최적화된 React와 초고속 빌드 도구인 Vite를 결합하였습니다.
*   **모바일 확장성**: 이 구조는 향후 Capacitor와 같은 도구를 사용하여 웹 코드를 모바일 앱 환경으로 변환할 때 최적의 성능을 낼 수 있는 가벼운 환경을 제공합니다.
