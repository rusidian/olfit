# 01. JavaScript vs TypeScript

### 1. JavaScript (JS) 개요
JavaScript는 웹 브라우저에서 동적인 기능을 구현하기 위해 사용되는 표준 프로그래밍 언어입니다. 사용자의 클릭 이벤트 처리, 데이터 통신, 화면 요소의 애니메이션 등 브라우저 내의 모든 동작을 제어하는 핵심 엔진 역할을 합니다.

### 2. TypeScript (TS) 개요
TypeScript는 JavaScript의 문법을 기반으로 '타입(Type)' 시스템을 추가한 언어입니다. 코드 작성 단계에서 변수나 함수의 데이터 형식을 명시적으로 정의함으로써, 자바스크립트의 자유로운 특성에서 발생할 수 있는 잠재적 오류를 사전에 차단합니다.

### 3. 프로젝트 도입 이유
*   **안정성 확보**: 실행 단계(Runtime)에서 발생할 수 있는 에러를 개발 단계(Build time)에서 미리 감지하여 서비스 중단 가능성을 최소화합니다.
*   **백엔드 통신 최적화**: Django DRF(Django REST Framework) 백엔드와 주고받는 JSON 데이터 구조를 '인터페이스(Interface)'로 정의하였습니다. 이를 통해 데이터 명칭 불일치나 누락으로 인한 통신 오류를 방지하고 프론트엔드와 백엔드 간의 데이터 정합성을 확보하였습니다.
